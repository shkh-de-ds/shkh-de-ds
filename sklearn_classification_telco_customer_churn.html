1. Data Exploration and Preparation¶

Content:¶

The IBM Sample Dataset has information about Telco customers and if they left the company within the last month (churn). Each observation represents a unique costumer, the columns contains information about customer’s services, account and socio data

The data set includes information about:¶
  • Customers who left within the last month – the column is called Churn
  • Services that each customer has signed up for – phone, multiple lines, internet, online security, online backup, device protection, tech support, and streaming TV and movies
  • Customer account information – how long they’ve been a customer, contract, payment method, paperless billing, monthly charges, and total charges
  • Demographic info about customers – gender, age seniority and if they have partners and dependents

# Lets look at how the monthly bill relates to churn
# We'll bin this attribute into categories

data_explore = data.copy()

def monthlyBill(data_explore) :
    
    if data_explore["MonthlyCharges"] <= 50 :
        return "$0-$50"
    elif (data_explore["MonthlyCharges"] > 50) & (data_explore["MonthlyCharges"] <= 75 ):
        return "$50-$75"
    elif (data_explore["MonthlyCharges"] > 75) & (data_explore["MonthlyCharges"] <= 100) :
        return "$75-$100"
    elif (data_explore["MonthlyCharges"] > 100) & (data_explore["MonthlyCharges"] <= 125) :
        return "$100-$125"
data_explore["MonthlyBill"] = data_explore.apply(lambda data_explore:monthlyBill(data_explore), axis = 1)

table = pd.crosstab(data_explore['MonthlyBill'],data_explore['Churn'])
